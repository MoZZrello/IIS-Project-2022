Implementace

Pre implementáciu bol použitý framework Django. Štruktúra a architektúra projektu je navrhnutá podľa doporučovaných konvencií v tomto nástroji. Django používa MVC, avšak Django Views zastupujú v prípade Djanga Controllers a Django templates sú Views. Templates sú html súbory uložené v priečinku templates, v ktorých prezentujeme dáta. Hlavnými časťami projektu sú súbory models.py, views.py, forms.py a urls.py, kde sa rieši url routing. Následne sme implementovali aj filtre v filters.py a kontrolovanie autentikácie užívateľa v decorators.py.

V models.py má na starosti dáta a databázovú schému, v ktorej sme si zadefinovali triedy, ktoré sú podtriedami models.Model a následne sme si pomocou dátových polí z nadtriedy vytvorili jednotlivé atribúty. Views.py je súbor, kde je spravená väčšina logiky IS. Pri implementácií sme využili funkcie, ktoré nám posytol framework Django. Funkcie v tomto súbore si môžeme rozdeliť na 2 podcelky.

Prvým je FunctionBasedView, ktorý má za úlohu prijať ako argument Http požiadavky (HttpRequest) a ako objekt vrátiť http odpoveď (HttpResponse).

Druhým je ClassBasedView, ktorý má na starosť prijatie konkrétneho požiadavku, jeho spracovanie a odoslanie odpovede.

Vo forms.py sú implementované jednotlivé formuláre ku views. Validácia prebieha automaticky pomocou frameworku. Zhromažďuje informácie o užívateľoch, ktoré sú potom odoslané na server.

Instalace

Prerekvizity pre lokálne spustenie:
  • Python 3.10
  • Pip
  • Django 4.1.1
  • Xampp pre MySQL databázu a Apache webserver
  • Pre Debian / Ubuntu príkaz:
    sudo apt-get install python3-dev default-libmysqlclient-dev build-essential
  • Pre Red Hat / CentOS príkaz:
    sudo yum install python3-devel mysql-devel
  • Následne položky v requirements.txt cez príkaz:
    pip install -r requirements.txt
Postup:
  1. Nainštalovanie prerekvizít
  2. Vytvorenie a nastavenie databázy iis_mysql
  3. Importovanie scriptu iis_mysql.sql
  4. Spustenie akcie apache a mysql v aplikácií xampp
  5. Sputenie serveru príkazom - python3 manage.py runserver
  6. Stránka je teraz dostupná na localhost adrese a porte 8000.
